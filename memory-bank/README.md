# Memory Bank

This directory stores compact, durable context for low-token Codex/Cline waves.

## Canonical Files

- `projectbrief.md`
- `productContext.md`
- `systemPatterns.md`
- `techContext.md`
- `activeContext.md`
- `progress.md`
- `context-pack.md`
- `decisions.md`
- `HANDOFF.md`
- `repoKitCatalog.md`
- `solutionHarvest.md`
- `commonPitfalls.md`

## Update Loop

1. Refresh active context with `scripts/memory/refresh_memory_bank.ps1`.
2. Review `memory-bank/repoKitCatalog.md` before each wave for repo-kit assets to reuse.
3. Record reusable solution candidates in `memory-bank/solutionHarvest.md`.
4. Record recurring failure/fix patterns in `memory-bank/commonPitfalls.md`.
5. Keep `context-pack.md` and `HANDOFF.md` compact for local AI context windows.
