# Build Order

## Purpose

The build order keeps the program coherent. Western foundations come first, comparative material follows through reviewed lanes, and learner-facing lessons come only after source and culture gates are ready.

## Phase 1: Western Foundation Spine

Build first:

1. Greek inquiry: Plato, Aristotle, Euclid.
2. Roman rhetoric and civic duty: Cicero, Quintilian.
3. Scholastic disputation: Abelard, Aquinas, Boethius.
4. Early modern inquiry: Bacon, Galileo, Descartes, Locke, Hume, Mill, Peirce.
5. Civic and moral judgment: Smith, Federalist Papers, Tocqueville, Lincoln, Douglass.

Deliverables:

- source packets for first Western texts
- lesson outlines
- drills
- assessments
- capstone examples

## Phase 2: Comparative Guardrails

Build before comparative lessons:

1. Civilization coverage map.
2. Culture difference framework.
3. Reviewer standard.
4. Source packet system.
5. Curriculum matrix.

Deliverables:

- comparative source list
- cultural caveats
- reviewer gates
- deferred items clearly marked

## Phase 3: Comparative Lesson Extensions

Add comparative material only after review:

1. Indian Nyaya and Buddhist logic.
2. Islamic falsafa, kalam, and legal reasoning.
3. Jewish legal/philosophical reasoning.
4. Chinese Confucian, Mohist, Daoist, and strategic traditions.
5. African and oral deliberative traditions where sources are strong.

Deliverables:

- original lesson notes
- contrast boxes
- transferable drills
- misread warnings
- reviewer notes

## Phase 4: Practice And Performance

Build practice systems:

1. proof reconstruction
2. disputation lab
3. speech lab
4. evidence audit lab
5. civic judgment lab
6. adversarial reasoning lab
7. teach-back lab

Deliverables:

- rubrics
- sample prompts
- revision paths
- performance pressure ladder

## Phase 5: Platform Integration

After content is stable, integrate with:

- `open-education-suite` for learner delivery
- `voice-studio` for spoken explanation and disputation practice
- Video Studio for lectures, board work, and dialogue reenactments
- Source Preservation Kit for source evidence packets
- QA Live for learner surface validation

## Build Rule

Do not build polished lessons from comparative traditions until source packets and cultural review exist. Build the Western spine first, then add comparative lanes as reviewed contrast and enrichment.
