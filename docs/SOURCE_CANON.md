# Source Canon

## Use Boundary

This repo records source references, learning purposes, and original exercises. It does not copy copyrighted translations or course assets.

When full texts are needed, use public-domain editions or properly licensed editions and preserve attribution. Public-domain status can vary by translation, editor, and country, so every future source packet must include edition and license evidence before text is copied.

## Core Classical Sources

| Source | Why It Matters | Use In Program | Boundary |
| --- | --- | --- | --- |
| Euclid, Elements | Definitions, postulates, common notions, construction, demonstration, proof sequence. | Proof craft, disciplined inference, cumulative reasoning. | Reference works and public-domain editions only unless license is explicit. |
| Aristotle, Categories | Terms, predicates, substance, quantity, quality, relation, and classification. | Definition discipline and category-error detection. | Reference concepts; do not copy modern translations without license. |
| Aristotle, On Interpretation | Propositions, contradiction, affirmation, denial, modality. | Claim structure, contradiction, and sentence-level logic. | Use public-domain translations or source references. |
| Aristotle, Prior Analytics | Syllogism and formal inference. | Validity, middle term, inference patterns. | Use examples and original exercises. |
| Aristotle, Posterior Analytics | Demonstration, first principles, explanatory knowledge. | Proof from causes, knowing why, first-principles reasoning. | Reference and original summaries. |
| Aristotle, Topics | Dialectical questioning, endoxa, places of argument. | Structured questioning, objection handling, debate practice. | Reference and original exercises. |
| Aristotle, Sophistical Refutations | Fallacies and deceptive reasoning. | Bad-faith argument detection and repair. | Reference categories; write original cases. |
| Aristotle, Rhetoric | Ethos, pathos, logos, audience, persuasion. | Ethical communication and public speaking. | Use public-domain editions or original paraphrase. |
| Plato, selected dialogues | Socratic questioning and intellectual humility. | Dialogue practice and question sequencing. | Use public-domain editions or excerpts with license evidence. |
| Cicero, De Inventione and De Oratore | Invention, arrangement, eloquence, public argument. | Speech craft and civic rhetoric. | Use licensed/public-domain editions. |
| Quintilian, Institutio Oratoria | Moral formation of the speaker and rhetorical training. | Character constraints and speaker discipline. | Use licensed/public-domain editions. |
| Boethius, logical works | Medieval transmission of logic and dialectic. | Bridge between ancient logic and later scholastic method. | Reference and licensed editions. |
| Classical trivium tradition | Grammar, logic, rhetoric as integrated formation. | Course spine for reading, reasoning, speaking. | Use as organizing model. |

## Later Great-Works Extensions

Future advanced tracks may draw from:

- Aquinas for disputation, objections, replies, distinctions, and synthesis.
- Bacon, Mill, and Peirce for induction, inquiry, and scientific reasoning.
- Locke and Whately for language, evidence, and rhetoric.
- Pascal and Newman for persuasion, conscience, probability, and assent.
- Lincoln, Douglass, Churchill, King, and other public speakers for rhetoric under moral pressure.

## Source Packet Requirements

Before adding actual source excerpts, create a source packet with:

- work title
- author
- edition or translation
- publication date
- license or public-domain evidence
- source URL or library reference
- excerpt boundary
- why the excerpt is pedagogically necessary
- attribution text
- last reviewed date

## Borrowing Rules

Allowed:

- cite source works
- summarize concepts in original words
- create original exercises based on source ideas
- link to public-domain or licensed editions
- quote short passages only when license and attribution are documented

Not allowed:

- copy modern translations without license
- copy paid course lectures, problem sets, or answer keys
- hide source uncertainty
- treat a source reference as proof of permission
