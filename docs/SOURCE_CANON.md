# Source Canon

## Use Boundary

This repo records source references, learning purposes, and original exercises. It does not copy copyrighted translations or course assets.

When full source passages are needed, use public-domain editions or properly licensed editions and preserve attribution. Public-domain status can vary by translation, editor, and country, so every future source packet must include edition and license evidence before text is copied.

## Integration Stages

| Stage | Meaning |
| --- | --- |
| `reference_only` | The work can be cited as an influence and used for original exercises, but no source text is copied here. |
| `packet_ready` | Edition, license/public-domain evidence, source URL or library reference, attribution, and excerpt boundary are documented. |
| `lesson_ready` | A course lesson, original exercises, assessment, and rubric are written against the source move. |
| `performance_ready` | Learners must use the source move under written, spoken, or adversarial pressure. |

The current repo keeps all listed works `reference_only` until source packets are created.

## Global And Comparative Extensions

A world-class reasoning course should not imply that only one civilization practiced disciplined thought. Comparative extensions may include:

- Nyaya logic and Indian debate traditions for inference, testimony, and debate structure.
- Buddhist logic and debate traditions for analysis of perception, inference, and refutation.
- Islamic philosophers such as Al-Farabi, Ibn Sina, and Averroes for logic, demonstration, and commentary traditions.
- Jewish philosophical and legal reasoning traditions for interpretation, objection, and reply.
- Chinese classical traditions for moral speech, order, prudence, and strategic reasoning.

These extensions require extra care because edition, translation, and cultural-context boundaries matter. Keep them reference-only until source packets and reviewer notes exist.

## Diagnostic And Adversarial Sources

Some works are useful mainly as warning labels.

- Schopenhauer's eristic dialectic can help identify manipulative debate tactics, but the program must train recognition and repair, not dishonest imitation.
- Machiavelli, Sun Tzu, and Clausewitz can train strategic awareness, adversarial incentives, and institutional realism, but must be tied to ethical constraints.
- Propaganda and demagoguery case studies should be handled as analysis targets, not templates.

## Source Packet Requirements

Before adding actual source excerpts, create a source packet with:

- work title
- author
- edition or translation
- publication date
- license or public-domain evidence
- source URL or library reference
- excerpt boundary
- why the excerpt is pedagogically necessary
- attribution text
- last reviewed date

## Borrowing Rules

Allowed:

- cite source works
- summarize concepts in original words
- create original exercises based on source ideas
- link to public-domain or licensed editions
- quote short passages only when license and attribution are documented

Not allowed:

- copy modern translations without license
- copy paid course lectures, problem sets, or answer keys
- hide source uncertainty
- treat a source reference as proof of permission
