# Program Design

## Mission

Build a course of study that turns clear reasoning into a normal way of seeing, speaking, reading, and acting. The learner should not merely know about logic or rhetoric. The learner should be able to use disciplined thought in real conversations, public arguments, writing, teaching, leadership, and self-correction.

## Working Name

`open-education-clear-reasoning` was chosen over narrower names such as `open-education-logic`, `open-education-trivium`, or `open-education-rhetoric` because the program combines logic, proof, dialectic, rhetoric, reading, speaking, and practical judgment.

The repo can still host named tracks such as:

- Trivium Track: grammar, logic, rhetoric.
- Euclid Track: definitions, axioms, constructions, proof.
- Aristotle Track: categories, propositions, demonstration, dialectic, rhetoric, ethics.
- Public Argument Track: debate, disagreement, fallacy resistance, and charitable steelmanning.

## Learner Outcomes

A graduate of the program should be able to:

1. Define terms and notice when a term shifts meaning.
2. Break an argument into claims, reasons, assumptions, warrants, examples, objections, and conclusions.
3. Distinguish deductive validity, inductive strength, explanatory power, rhetorical force, and moral persuasion.
4. Reconstruct another person's argument so fairly that the person recognizes it.
5. Build proof-like explanations from definitions, accepted premises, and visible inferences.
6. Identify contradiction, equivocation, circularity, category mistakes, false dilemmas, weak analogies, non sequiturs, and manipulative framing.
7. Ask ordered questions that expose structure rather than merely score points.
8. Speak clearly under friendly, confused, hostile, or bad-faith pressure.
9. Use rhetoric ethically: fitting words to audience, purpose, evidence, and moral constraints.
10. Transfer the method to politics, family, work, faith, science, media, education, and self-government.

## Course Architecture

The first course is `Clear Reasoning Foundations`.

It has twelve modules:

1. Attention, intellectual posture, and the habit of distinction.
2. Terms, definitions, categories, and predication.
3. Propositions, contradiction, implication, and consequence.
4. Euclidean proof craft: definitions, postulates, common notions, construction.
5. Syllogistic reasoning and valid inference.
6. Demonstration, first principles, and explanatory knowledge.
7. Dialectic: questions, objections, concessions, and ordered inquiry.
8. Fallacies, sophistry, and manipulative argument styles.
9. Rhetoric: audience, ethos, pathos, logos, arrangement, and style.
10. Dialogue and disagreement: steelmanning, charity, courage, and friendship.
11. Modern transfer: media claims, policy debates, institutions, data, and incentives.
12. Capstone: proof portfolio, live disputation, speech, and self-audit.

## Mastery Model

The course uses five mastery bands:

- `recognize`: learner can identify a pattern when prompted.
- `name`: learner can name and define the pattern precisely.
- `analyze`: learner can break a real example into structure.
- `produce`: learner can build a clear proof, argument, objection, or speech.
- `perform`: learner can do it under live pressure while preserving accuracy and character.

## Practice System

Every module includes:

- close reading from a source or source excerpt reference
- definition drill
- argument mapping drill
- proof or explanation drill
- objection and reply drill
- speech or dialogue drill
- reflection and transfer exercise
- summative assessment with rubric

## World-Class Bar

A unit is not complete unless it includes:

- precise objective statements
- great-work source anchor
- modern transfer case
- practice that can be repeated daily
- assessment requiring explanation, not merely recall
- speaking component
- error-detection component
- ethical constraint
- instructor or self-review rubric
- public-safe provenance and license notes

## Non-Goals For First Scaffold

The initial repo does not implement a learner UI, adaptive engine, generated lecture video, or live debate simulator. Those belong in later waves after the curriculum contract is stable.
