# Reasoning Drills

## Drill 1: Term Lock

1. Pick one disputed term.
2. Write a working definition.
3. Name the genus and distinguishing feature if possible.
4. Give two clear examples.
5. Give two borderline cases.
6. Give one thing the term does not mean.
7. Re-read the argument and mark every place the term shifts.

## Drill 2: Claim Stack

1. Split a paragraph into individual claims.
2. Label each claim as definition, observation, causal claim, value claim, prediction, inference, or conclusion.
3. Identify which claims need evidence.
4. Identify which claims are doing the most work.
5. Rewrite the argument with the claims in logical order.

## Drill 3: Euclidean Step Test

1. State the starting definitions or assumptions.
2. State the desired conclusion.
3. Write each step on a separate line.
4. For each step, ask: from what previous step does this follow?
5. Mark missing premises.
6. Remove decorative rhetoric and see whether the proof still stands.

## Drill 4: Syllogism Translation

1. Find a claim that sounds persuasive.
2. Translate it into major premise, minor premise, and conclusion.
3. Identify the middle term.
4. Test whether the conclusion follows.
5. If invalid, repair the form or weaken the conclusion.

## Drill 5: Steelman Ladder

1. State the opposing view as opponents usually say it.
2. Remove insults, slogans, and weak examples.
3. Name the strongest concern behind it.
4. State the view so fairly that a serious opponent would accept it.
5. Answer only after completing the fair version.

## Drill 6: Fallacy Repair

1. Name the suspected fallacy.
2. Quote or summarize the exact move.
3. Explain why it fails.
4. Rewrite the argument into the strongest valid or plausible version.
5. Decide whether the repaired argument is still persuasive.

## Drill 7: Question Ladder

Ask questions in this order:

1. What do you mean by the key term?
2. What claim are you making exactly?
3. What reason supports it?
4. What would count against it?
5. What standard are we both using?
6. What follows if you are right?
7. What follows if you are wrong?

## Drill 8: Speech Compression

1. Explain the argument in five minutes.
2. Explain it in one minute.
3. Explain it in one sentence.
4. Add back only what a real listener needs.
5. Check whether compression changed the claim.

## Drill 9: Combative Opponent Reset

When the other person attacks instead of argues:

1. Name the claim you are willing to answer.
2. Refuse the insult without returning it.
3. Ask for the standard of proof.
4. Answer the strongest claim, not the noise.
5. End the exchange if the person will not return to reasons.

## Drill 10: Self-Audit

After any argument, write:

- where I defined terms well
- where I shifted terms
- where I overstated evidence
- where I represented the other person fairly
- where I caricatured or dodged
- what I will practice before the next discussion
