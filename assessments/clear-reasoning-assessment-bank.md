# Clear Reasoning Assessment Bank

CR-101 favors essays, spoken explanations, written dialogues, repair packets, and defended portfolios because clear reasoning mastery requires synthesis, performance, transfer, and revision.

## Assessment Types

- Diagnostic essay: diagnose a confused argument and repair it respectfully.
- Definition packet: stabilize terms and test borderline cases.
- Proof reconstruction: rebuild a proof-like argument with ordered steps.
- Fallacy repair packet: identify the error, steelman the strongest version, and revise.
- Dialogue performance: ask ordered questions and answer objections.
- Capstone defense: defend and revise a reasoning portfolio.

## Rubric

Strong work:

- cites objective IDs and argument evidence
- defines terms before arguing
- separates claim, evidence, warrant, assumption, and conclusion
- represents opponents fairly
- repairs weak reasoning into a stronger form
- revises after feedback

Weak work:

- attacks a weaker version of the opposing view
- argues from slogans or tone alone
- changes definitions midstream
- labels a fallacy without explaining the repair
- treats persuasion as manipulation
